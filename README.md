# freefall-linear-regression-analysis
Multiple linear regression pipeline in Python to model freefall times from experimental physics data for spheres of different densities. I compared MSE/MAE loss functions and batch, stochastic, and mini-batch gradient descent approaches.

Assignment and report were submitted as part of PHYS20035: Computational Physics and Data Science at the University of Bristol — awarded 78%
